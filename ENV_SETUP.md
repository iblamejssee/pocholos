# Variables de Entorno - Pocholo's Chicken POS

Copia este archivo a `.env.local` y completa con tus credenciales de Supabase.

```bash
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=tu_url_de_supabase_aqui
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_anon_key_de_supabase_aqui
```

## CÃ³mo obtener las credenciales:

1. Ve a [Supabase Dashboard](https://supabase.com/dashboard)
2. Selecciona tu proyecto
3. Ve a Settings > API
4. Copia la URL del proyecto y la clave anon/public
